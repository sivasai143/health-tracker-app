#  HealthTracker — React Native Health App

A clean, feature-rich personal health tracking app built with React Native and Expo.

![React Native](https://img.shields.io/badge/React_Native-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)
![Expo](https://img.shields.io/badge/Expo-000020?style=for-the-badge&logo=expo&logoColor=white)

## Features

- **Dashboard** – Visual overview of today's health metrics with progress bars
-  **Daily Log** – Log steps, water intake, sleep, calories & mood
-  **History** – View past entries in a clean card layout
-  **Goals** – Track daily health goals with completion status
- **Weekly Chart** – Bar chart of weekly activity
-  **Mood Tracker** – Log daily mood with emoji selector



### Prerequisites
- Node.js (v18+)
- Expo CLI

### Installation

```bash
git clone https://github.com/YOUR_USERNAME/health-tracker-app.git
cd health-tracker-app
npm install
npx expo start
```

Scan the QR code with **Expo Go** app on your phone.
